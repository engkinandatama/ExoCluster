# References

This project builds upon several key scientific and computational tools.

## Core Scientific Tools

### Biosynthetic Gene Cluster (BGC) Prediction
- **BGC-Prophet**: Deep learning-based BGC annotator and classifier.
  - [GitHub Repository](https://github.com/HUST-NingKang-Lab/BGC-Prophet)

### Protein Language Models
- **ESM2 (Evolutionary Scale Modeling)**: Large-scale protein language models by Meta AI.
  - [GitHub Repository](https://github.com/facebookresearch/esm)

### Bioinformatics Utilities
- **Biopython**: Tools for biological computation.
  - [Official Website](https://biopython.org/)
- **MMseqs2**: High-throughput protein sequence searching and clustering.
  - [Official Website](https://github.com/soedinglab/MMseqs2)

## Workflow & Infrastructure

- **Snakemake**: Workflow management system for computational pipelines.
  - [Documentation](https://snakemake.readthedocs.io/)
- **Conda / Bioconda**: Package and environment management for bioinformatics.
  - [Bioconda Channel](https://bioconda.github.io/)
- **DVC (Data Version Control)**: For managing large model weights and datasets.
  - [Official Website](https://dvc.org/)

## Machine Learning Frameworks

- **PyTorch**: Deep learning library.
  - [Official Website](https://pytorch.org/)
- **Scikit-learn**: Machine learning and statistical modeling.
  - [Official Website](https://scikit-learn.org/)
